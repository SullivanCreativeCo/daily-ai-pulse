LI . NO. 03
Google just made cheap agents capable of doing actual computer work.

Gemini 3.5 Flash now has computer use baked in. Click, scroll, type, across browser and desktop environments. Not as an add-on. Built directly into the lightweight model.

That matters because the old math was: computer use requires a big expensive model, which limits how many tasks you can run at scale.

New math: the fast affordable model does the clicking. Sustained multi-step automation without the cost blowing up.

For small business owners with repetitive software tasks, this is worth paying attention to right now. The tools to automate the stuff you're manually clicking through every day are genuinely affordable.

If it's repeatable, it's a skill. And if it's a skill, you can teach an agent to do it.

New tools & harnesses

  • Stably Orca - aiinsiders.net An open-source IDE for running fleets of coding agents in parallel. Each agent gets its own isolated git worktree. You fan one prompt across five agents, each produces a different implementation, you pick the winner. Not one agent you're babysitting: five agents competing, you're the judge. Supports 30+ CLI agents including Claude Code, Codex, Cursor, Grok, and Devin. MIT licensed. If you're building software with agents and not running them in parallel yet, this is the tool that makes that workflow real. https://aiinsiders.net/article/stably-ships-orca-an-open-source-ide-built-for-fleets-of

  • Vercel Eve - infoq.com An open-source framework for building AI agents in production. Filesystem-first: instructions live in Markdown, tools are TypeScript files, subagents and skills are directories. It handles durable execution, sandboxing, human approval workflows, and evaluation without you building that scaffolding yourself. The whole point is: stop rebuilding the same plumbing every time you spin up a new agent project. https://www.infoq.com/news/2026/06/vercel-eve-agents/

  • Relume for site structure - emplex.nl Not a page builder. A planning tool. You describe a business, Relume generates a full sitemap and wireframes with draft copy that fits the actual project. It compresses the slow part of web design: deciding which pages exist and what they say. One hour instead of a week. If you're building sites for local clients and want to cut planning overhead, this one's worth knowing about. https://emplex.nl/relume-ai-sitemaps-and-wireframes/
