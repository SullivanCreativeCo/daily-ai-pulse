  • Thinking Machines Lab ships first model with 200ms interaction — Unite.AI
    Mira Murati's startup ended a year of silence with TML-Interaction-Small, a 276B-parameter MoE system (12B active) trained from scratch to process audio, video, and text in 200-millisecond chunks instead of waiting for turn-taking. The company raised $2B at a $12B valuation without shipping anything until now. The model preview signals a bet that latency, not capability, is the next competitive moat. https://www.unite.ai/thinking-machines-lab-ships-first-model-with-200ms-real-time-interaction/

"The first product from a lab that has raised about $2 billion at a $12 billion valuation without shipping anything beyond a fine-tuning tool."

  • Google unveils Googlebook: Android-powered AI laptops replace Chromebooks — The Next Web
    Google killed ChromeOS and unveiled Googlebook, a premium Android laptop category with Gemini embedded at the OS level. The devices ship this fall from Acer, ASUS, Dell, HP, and Lenovo. The most revealing feature: Magic Pointer, built with DeepMind, turns the cursor into an AI agent. No containers, no emulation. The OS is Android. The AI is the interface. ChromeOS is not being updated. It's being replaced.
    https://thenextweb.com/news/google-killed-the-chromebook-its-replacement-turns-your-cursor-into-an-ai-agent

  • Microsoft ships Foundry Local 1.1 with live transcription, embeddings, responses API — Microsoft Foundry Blog
    Microsoft's cross-platform local AI solution added live audio transcription for real-time speech-to-text, text embeddings for semantic search and RAG, and Responses API support for structured agentic interactions including tool calling and multimodal vision-language input. No cloud dependency. No network latency. No per-token costs. The WebGPU execution provider now ships separately to reduce default package size. The product thesis: local inference is table stakes for enterprise deployment.
    https://devblogs.microsoft.com/foundry/foundry-local-v1-1/

  • OpenAI launches Daybreak to take on Anthropic's Mythos in cyber defence — The Next Web
    OpenAI unveiled Daybreak, a cybersecurity initiative pairing GPT-5.5 variants with Codex Security and enterprise security partners. The platform finds vulnerabilities, generates patches, and validates fixes inside enterprise codebases. Three model variants: GPT-5.5 for general use, GPT-5.5 with Trusted Access for Cyber for defenders performing secure code review and patch validation, and GPT-5.5-Cyber for authorized red teaming. The goal: compress security analysis from hours into minutes with audit-ready evidence. This positions OpenAI directly against Anthropic's Mythos, which has dominated AI-powered defence conversation for months.
    https://thenextweb.com/news/openai-daybreak-anthropic-mythos-cyber-defence
