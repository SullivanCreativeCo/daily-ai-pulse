Today's headline

SpaceX bought Cursor for $60 billion Tuesday morning. Same morning, Grok V9-Medium started its public rollout.

Not a coincidence. Over a million developers chose Cursor because it ran Claude, GPT, and other models simultaneously. Whether that stays true after SpaceX takes full ownership is the question nobody has a clean answer to yet.

SpaceX just bought it for $60 billion.

LI . NO. 02
One-sixth the cost. Same or better performance on coding benchmarks.

Z.ai dropped GLM-5.2 yesterday. 753 billion parameters, open weights, MIT license. Claims to outperform GPT-5.5 at roughly 16 cents on the dollar. That's not a modest pricing improvement. That's a different cost category entirely.

The "AI is too expensive to run" objection has a shorter shelf life every week.

I keep thinking about builders and small business owners sitting on ideas because the math didn't work. That math is changing. Fast. The cost curve on inference isn't stabilizing, it's accelerating downward.

It's not about making millions with one idea. It's about having several profitable little ideas. When the infrastructure cost keeps dropping like this, more of those ideas start working.

What would you build if inference was close to free? That's the question worth sitting with today.
LI . NO. 03
Everyone's building agents. Nobody's building them the same way twice.

Vercel shipped "eve" today, an open-source agent framework. The pitch: agents are where the web was before frameworks. Everyone hand-rolling the same infrastructure on every new project. Nothing carrying over. Every agent starts from scratch.

Eve comes with durable execution, sandboxed compute, and human-in-the-loop approvals already built in. An agent is literally a directory. Instructions file. Tools folder. Skills. Schedules. Each piece in its own place.

They're comparing it to what Next.js did for the web. Big claim. But if you've ever started a second agent project and caught yourself rebuilding all the same plumbing, the problem they're naming is real.

The framework era for agents just started. That's the marker to notice, not the specific tool.
